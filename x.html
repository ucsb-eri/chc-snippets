EWXx Map Snippet Configuration

 
Time navigation

Snippets are cofigured by adding the following lines to an HTML file. Copy the line of code below [ and paste it above the intended location of the first EWX map in your web page. Make sure this line appears only once in your page.

<!DOCTYPE html>
<html>
  <head>
  </head>
  <body>
    <script>
    new EwxMap({
      id: 'ewx-map-h4vbt36xss',
      dataset: 'CHIRPSGEFS',
      subDataset: 'chirps_gefs',
      region: 'global',
      periodicity: '1-day',
      forecastPeriod: '15day',
      period: 'latest',
      showDekadControl: true,
      showYearControl: true,
      units: 'mm',
      statistic: 'data',
      width: 790,
      height: 790,
      zoom: 3.5,
      center: [6.0, 17.0],
    });
    </script>
  </body>
</html>
