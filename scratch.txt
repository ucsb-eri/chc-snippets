=========== Snippets Talk ==================

Snippets are a way of esentially embedding an EWX map window in any web page.

Snippets can display any of the datasets that are available in our EWX

With modification, other EWX installations can be used.

Viewers can pan and zoom the maps and change dates to view historical data.

----

The web developer only needs to add a few lines of Javascript code to an HTML page to install.

Download the JS contoller...
<script src="https://snippets.chc.ucsb.edu/ewxmapG5.js"></script>

Add a div tag to place the Snippet on the web page
<div id="snippet-m08f05L58"></div>

Add the configuration code to focus the Snippet on the ROI
<script> type="text/javascript"
  new EwxMapG5({
    ewxUrl: src='https://ewx3.chc.ucsb.edu/ewx',
    id: 'snippet-m08f05L58',
    dataset: 'CHIRPS',
    subDataset: 'chirps',
    region: 'global',
    periodicity: 'dekad',
    period: 'latest',
    showDekadControl: 'true',
    showYearControl: 'true',
    units: 'mm',
    statistic: 'data',
    width: 800,
    height: 600,
    zoom: 5.5,
    center: [3.8, 42.3],
  });
  </script>

The use can modifiy and install this code by using our Configuration Documentation
  https://snippets.chc.ucsb.edu/configuration.html
