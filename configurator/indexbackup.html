Introduction

Snippets are a free, publicly accessible tool to display maps from the Climate Hazards Center Early Warning Explorer (EWX) online application on your own webpage. Snippets access the Climate Hazards Center The CHC maintains extensive data stores that are regularly updated.

The Climate Hazards Center, as a part of FEWS NET, has vast information stored in the data repository and in the Early Warning eXplorer . And yet, the most important decisions are local. The Snippets were developed to allow national, meteorological, hydrological, and agricultural agencies to easily set up localized web-mapping services. We hope that Snippets can help inform proactive decisions that enhance resilience.

Configure Snippet code

To configure your CHIRPS-GEFS EWX Map Snippet, enter a unique Snippet ID and select the desired Periodicity and statistic of interest. Enter the Width and Height of your Snippet, the Center Latitude and Longitude and the Zoom level of the map. You can also choose to enable or disable the time navigation controls of periodicities and years for the Snippet. Then click the Preview Snippet button to view the Snippet as it would appear in your web page. Close the preview page and make any adjustments and repeat as neccesary. When you have configured the Snippet to your satisfaction, click the Create Configuration button to generate the HTML code for the Snippet.

Configure your web page with Snippet code

Copy and paste the first or second line of code and paste it above the intended location of the first EWX map in your web page. You will need to copy the ewxmapG5.js file to the same directory as your web page to use the first line. Make sure this line appears only once in your page. Place the <div> line at the location you want the Snippet to appear. Note that the Snippet ID must be unique for each Snippet on the web page.
